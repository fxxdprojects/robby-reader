# robby-reader
A lightweight, tab-based PDF reader for Linux built with Python, PyQt5, and Poppler.

## Features
- **Tabbed Interface**: Keep all your PDFs in one window.
- **Session Restoration**: Remembers your open tabs and scroll positions even after a restart.
- **Deep Search**: Case-insensitive search with native and fallback extraction.
- **Recent Files**: Quick access to your last 10 documents.
- **Shortcuts**: Full keyboard support (Ctrl+O, Ctrl+F, etc.)

## Installation
1. Install system dependencies:
   `sudo apt install python3-pyqt5 python3-poppler-qt5`
2. Clone this repo:
   `git clone https://github.com/YOUR_USERNAME/robby-reader.git`
3. Run it:
   `python3 robby_reader.py`
